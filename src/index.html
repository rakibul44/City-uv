IT HelpDesk Process

An example process about how HelpDesk work like.

Submit A ticket

You can submit a ticket from this home page or dashboard after login. If you don't have login access, you can create an account from this link.

Instant talk with agent

You can discuss with the agent directly through the "let's talk" button, if your ticket is being delayed for response.

Track progress with email

You will get email updates when the ticket status changes, or you can comment and discuss with the agent for a particular ticket discussion.

Done and Close the ticket

After marking a ticket as done, the agent will close the ticket. You may receive a notification when the ticket is closed.
